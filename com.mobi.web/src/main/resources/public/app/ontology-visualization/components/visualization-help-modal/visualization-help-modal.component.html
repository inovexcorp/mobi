Visualization Help

Visualization Graph

  • Nodes in the visualization graph can be clicked to highlight them and focus their entry in the sidebar
  • Nodes can be dragged to reposition them manually
  • Nodes are color-coded based on their source ontology
  • Dotted lines represent subclass-of relationships
  • Solid lines represent object properties and display a label indicating the relationship type
  • A maximum of 500 nodes will be rendered in the visualization graph. A snackbar warning is shown on initial load if this limit is exceeded.

Zoom

  • Use the mouse scroll wheel or the zoom bar to zoom in and out of the visualization graph
  • Right-click a class in the sidebar to center the visualization graph on that class

Sidebar

  • Use the search box to filter classes by name
  • Toggle between local and imported ontologies using the ontology filter
  • All classes are grouped by source ontology and sorted alphabetically
  • Checkboxes allow you to show or hide specific ontologies/classes
  • Clicking a class in the sidebar highlights the corresponding node in the visualization graph
